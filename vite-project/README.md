#  Gerenciador de Tarefas
Este projeto consiste no desenvolvimento de uma aplicação web utilizando **HTML, CSS e JavaScript**, com o objetivo de simular um sistema simples de gerenciamento de tarefas.

A aplicação permite cadastrar, visualizar e gerenciar tarefas, atendendo aos requisitos propostos na atividade prática da disciplina.


##  Objetivo

Desenvolver uma aplicação frontend que contemple:

* Navegação entre múltiplas páginas
* Formulário com validação
* Listagem dinâmica de dados
* Gerenciamento de estado em memória
* Interface responsiva

## Funcionalidades

✔ Cadastro de tarefas com validação de campos
✔ Armazenamento de dados utilizando `localStorage`
✔ Listagem dinâmica das tarefas cadastradas
✔ Filtro por status (Todas, Pendentes e Concluídas)
✔ Marcar tarefas como concluídas
✔ Exclusão de tarefas
✔ Dashboard com resumo (total, pendentes e concluídas)
✔ Gráfico de visualização de tarefas (Chart.js)

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js (para visualização de dados)

## Como Executar o Projeto

1. Clone o repositório:

git clone https://github.com/Guilhene/lista_Task/vite-project.git

2. Acesse a pasta do projeto:

cd lista_Task

3. Abra o arquivo `index.html` no navegador.
